Рисует лужи

В оределенных координатах на дороге рисуются три лужи разного размера. При продолжительности дождя лужи увеличиваются, затем уменьшаются (высыхают).

Аргументы
t- время увеличения/уменьшения лужи в размерах +t - лужа увеличивается, -t - лужа уменьшается
size- размер лужи в начальный момент
Заметки
Лужа рисуется в виде элипса и может увеличиваться и уменьшаться При уменьшении лужи необходимо задать ее "начальный" размер, чтобы она не уменьшилась меньше нуля и элипс не начался рисовать в обратном направлении (позволило не использовать if в функции).
Пример использования:

Рисует окна в доме

Рисуются окна определенного размера в несколько рядов

Аргументы
x0- координата левой верхней точки окна по оси х
y0- координата левой верхней точки окна по оси у
x1- ширина окна (координата правой нижней точки окна по оси х)
y1- высота окна (координата правой нижней точки окна по оси у)
count- задает количество рядов окон в доме
Заметки
Окна рисуются исходя из количества рядов count. Каждый ряд состоит из пяти окон между которыми задается ширина и высота с помощью внутренних переменных size_width и size_height.
Пример использования:
drawWindows (x, y, ROUND(x + 150*sizeX), ROUND(y + 200*sizeY), windows);
Предупреждения
Функция рисует только окна и поэтому вызывается из функции рисования дома.
